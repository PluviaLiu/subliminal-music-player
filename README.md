# subliminal-music-player
A website which plays subliminal music by inputting .mp3 files directly.


# 多轨音频播放器

一个用于叠加播放潜意识音频的小工具。无需安装，直接双击即可在浏览器中运行。

## 背景

制作潜意识视频需要将潜意识音频与背景音乐或白噪音叠加，并同步控制各轨道的循环节奏。

传统做法是在剪辑软件中手动导入、对齐、导出，流程繁琐。该播放器将这个监听与调试阶段的工作流提取到浏览器内完成，方便实时调节各轨参数。

## 功能

### 模块1:潜意识音频模块（多轨混音）

- 同时播放任意数量的潜意识音频，轨道之间互不干扰
- 每条轨道可独立控制，播放/停止
- 默认开启单轨循环（`audio.loop = true`），即自动循环播放
- 支持一次导入多个文件

### 模块2:音乐模块（切歌播放器）

- 同一时间只有一首歌处于播放状态，切换时自动停止上一首
- 点击列表中的曲目即可切换；再次点击当前曲目则播放 / 暂停
- 底部 transport 栏支持上一首 / 下一首 / 拖拽进度
- 循环模式可在「单次 → 单曲循环 → 列表循环」之间切换

## 使用方法

```
1. 下载 player.html
2. 用任意现代浏览器（Chrome / Safari / Firefox / Edge）打开
3. 在「潜意识音频模块」中点击「＋ 添加音轨」，导入 MP3 / WAV 文件
4. 在「音乐模块」中点击「＋ 添加音乐」，导入背景音乐
5. 分别点击各轨道的播放按钮开始混听
```

所有文件均在本地加载，不产生任何网络请求，无隐私风险。

## 技术说明

- 纯原生实现：HTML5 / CSS3 / Vanilla JavaScript，无任何外部依赖
- 音频通过 `URL.createObjectURL()` 在内存中加载，关闭页面后自动释放
- 潜意识模块每条轨道对应一个独立的 `Audio` 实例，天然并行
- 音乐模块共用单一 `Audio` 实例，切歌时先 `pause()` 再替换 `src`，确保互斥

## 已知限制

- 播放列表不持久化，刷新页面后需重新导入文件（浏览器安全沙箱限制）
- 不支持直接导出混音后的音频文件（如需导出，建议使用系统录音或 DAW）
- 移动端浏览器可能因自动播放策略要求用户先触发交互
